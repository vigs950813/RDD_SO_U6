Glosario

    • Algoritmo

      Conjunto de pasos o reglas definidos y ordenados que describe el proceso a seguir para realizar una tarea o resolver un problema.

    • API (Interfaz de Programación de Aplicaciones)

      Conjunto de reglas y herramientas que permite la comunicación entre diferentes aplicaciones de software. Facilita la interacción y el intercambio de datos entre sistemas, posibilitando el desarrollo de nuevas funcionalidades o la integración de servicios.

    • Bit

      Unidad de información más pequeña en informática. Un bit puede tener uno de dos valores posibles: 0 o 1, que son los componentes básicos del sistema binario utilizado por las computadoras para representar y procesar información.

    • CLI

      Interfaz de Línea de Comandos. Es uno de los principales medios que permiten interacción entre usuario y computadora, su función es ejecutar instrucciones a través de los comandos que se le proporcionan.

    • CPU

      Unidad Central de Procesamiento. Componente clave de un sistema informático que ejecuta instrucciones de un programa almacenado en memoria y realiza operaciones aritméticas y lógicas.

    • ext4 (Cuarto Sistema de Archivos Extendido)

      Evolución del sistema de archivos ext3 y es conocido por su alto rendimiento, robustez y soporte para discos y archivos de gran tamaño.

    • FAT

      Familia de sistemas de archivos utilizados para organizar y gestionar la información en medios de almacenamiento

    • Hardware

      Conjunto de componentes físicos que forman una computadora o sistema informático, como la CPU, memoria, disco duro, etc.

    • Kernel

      Núcleo central de un sistema operativo que gestiona los recursos del hardware y proporciona servicios básicos para el funcionamiento de las aplicaciones. Controla el acceso a la memoria, la administración de procesos y la comunicación entre software y hardware.

    • KiloByte

      Unidad de medida de almacenamiento de información que equivale a 1,024 bytes. Aunque el prefijo "kilo-" en el Sistema Internacional de Unidades (SI) normalmente significa 1,000, en la informática tradicionalmente ha significado 1,024 porque las computadoras operan en base binaria, y 1,024 es una potencia de 2 (2¹⁰ = 1,024).

    • Linux

      Sistema operativo de código abierto basado en el núcleo Linux, desarrollado originalmente por Linus Torvalds en 1991.

    • macOS

      Sistema operativo desarrollado por Apple para sus computadoras Mac. Ofrece una interfaz gráfica intuitiva y funcionalidades optimizadas para el hardware de Apple, como el MacBook y iMac. Basado en Unix, destaca por su estabilidad, seguridad y un ecosistema integrado con otros dispositivos de Apple, como iPhone y iPad.

    • Memoria

      Dispositivo o sistema que retiene y almacena datos para su uso posterior, incluida la memoria RAM (memoria de acceso aleatorio) y la memoria de almacenamiento a largo plazo.

    • MMU (Unidad de Gestión de Memoria)

      Componente de hardware que se encuentra dentro de la CPU o como un chip separado y que se encarga de gestionar y traducir las direcciones de memoria virtual a direcciones físicas. Es esencial para la implementación de la memoria virtual en los sistemas operativos modernos.

    • NTFS

      Sistema de Archivos de Nueva Tecnología. Es el sistema de archivos estándar para los sistemas operativos Windows desde Windows NT 3.1, incluyendo todas las versiones posteriores como Windows 2000, XP, Vista, 7, 8 y 10, así como los servidores Windows.

    • Overhead

      Carga adicional o recursos utilizados por un sistema operativo o programa que no contribuyen directamente a la tarea principal. En sistemas operativos, el overhead puede manifestarse en términos de consumo de memoria, tiempo de procesamiento o recursos del procesador. Reducir el overhead es fundamental para optimizar el rendimiento del sistema y mejorar la eficiencia en la ejecución de tareas.

    • Planificación

      Actividad que involucra la definición de metas, la determinación de los recursos necesarios y la organización de las acciones a seguir para lograr un objetivo específico.

    • Proceso

      Programa en ejecución, la instancia activa de un programa en la memoria de la computadora, con sus recursos asignados, incluido el espacio de memoria, los archivos abiertos y otros datos.

    • Quantum

      Intervalos o unidades discretas indivisibles de tiempo.

    • RAM

      Memoria de Acceso Aleatorio. Componente físico de cualquier computadora, se caracteriza por ser un dispositivo volátil, lo que significa que permite almacenar información de manera temporal.

    • ROM

      Memoria de Solo Lectura. Es un tipo de memoria que se utiliza principalmente en dispositivos electrónicos para almacenar datos que no deben ser modificados o que deben mantenerse permanentemente incluso cuando el dispositivo se apaga.

    • Shell

      Interfaz que proporciona un entorno para que los usuarios interactúen con el sistema mediante comandos. Puede ser una interfaz de línea de comandos (CLI) o una interfaz gráfica de usuario (GUI). El shell interpreta los comandos del usuario y los traduce para que el sistema operativo los ejecute. Además de actuar como intérprete de comandos, algunos shells también ofrecen funciones adicionales, como scripting y automatización de tareas.

    • Software

      Conjunto de programas, instrucciones y datos que controlan el funcionamiento de un sistema informático. Incluye aplicaciones y el sistema operativo.

    • Sincronización

      Coordinación y control temporal de las operaciones en un sistema, especialmente en entornos multiusuario o multitarea, para evitar conflictos y garantizar la consistencia de los datos.

    • Swap/Swapping (Intercambio)

      Técnica de gestión de memoria que consiste en mover temporalmente datos o procesos de la memoria principal (RAM) a un espacio de almacenamiento secundario (generalmente en el disco duro) para liberar memoria y permitir que otros procesos puedan ejecutarse.

    • Swap space (Espacio de intercambio)

      Porción del disco duro que el sistema operativo utiliza como una extensión de la memoria RAM cuando esta se agota. Es una región reservada en el almacenamiento secundario donde se pueden mover temporalmente los datos o procesos que no se están utilizando activamente en la memoria principal.

    • UNIX

      Sistema operativo multitarea y multiusuario desarrollado en los años 70 por AT&T en los laboratorios Bell.

    • Windows

      Sistemas operativo más popular y ampliamente utilizado en el mundo, especialmente en computadoras personales. Desde su primera versión lanzada en 1985, ha evolucionado considerablemente, con interfaces gráficas intuitivas y soporte para una amplia variedad de aplicaciones.
